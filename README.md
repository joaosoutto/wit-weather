# Wit Weather Project

## Getting Started

Clone repositorie

```bash
git clone git@github.com:joaosoutto/wit-weather.git
# or
git clone https://github.com/joaosoutto/wit-weather.git
```

Enter in project's folder

```bash
cd wit-weather
```

Install the dependencies

```bash
npm install
```

Then run it!

```bash
npm start
```
